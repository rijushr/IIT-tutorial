My First Web Page

In this module, the key learning expectation is to develop a strong foundation in HTML web programming by understanding its fundamental concepts and structure. You will learn how HTML serves as the backbone of web development, how elements are organised, and how markup languages are used to structure content effectively. This will include exploring the syntax of HTML instructions and the rules that govern their use, enabling you to create clear and well-structured web pages.
By the end of the module, you are expected to confidently design and build a basic website using HTML. You will gain practical skills in applying colour, incorporating graphics, and modifying visual properties to enhance the appearance and functionality of web pages. These skills will not only help you understand the essential role of HTML in web programming but also prepare you for more advanced concepts in web development.
